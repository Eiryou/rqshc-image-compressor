# プライバシー方針 / Privacy Policy

最終更新日 / Last updated: 2026-07-13

## 日本語

RQSHC V64IとRQI Viewerは、画像を利用者のWindows PC内で処理するローカル優先のソフトウェアです。画像を圧縮・閲覧するために、外部のクラウドサービスへ意図的に送信しません。

アプリケーションは、次の情報を独自の永続データとして保存しない方針です。

- 入力画像
- 画像キュー
- サムネイル
- 閲覧履歴
- 入力ファイル名やローカルパスを含む履歴
- 利用者が明示的に保存していない圧縮・復元結果

Viewerは、選択された日本語／Englishの表示設定だけを保存します。この設定に画像、ファイル名、パスは含まれません。

利用者が保存、PNG書き出し、クリップボードへのコピーを選んだ場合、その出力は指定した保存先またはWindowsのクリップボードへ渡されます。

Windowsのページファイル、休止ファイル、クラッシュダンプ、ブラウザ、ウイルス対策ソフト、バックアップ製品、クリップボード履歴など、本ソフトウェアの管理外の機能が一時的な痕跡を保持する可能性があります。あらゆる痕跡が絶対に残らないことは保証しません。

プライバシー上の問題を報告するときは、公開Issueへ個人画像、ファイル名、ローカルパス、機密ログを添付しないでください。

## English

RQSHC V64I and RQI Viewer are local-first applications that process images on the user's Windows PC. They are not designed to intentionally upload source images to an external cloud service for compression or viewing.

The applications are designed not to persist the following as application data:

- Source images
- Image queues
- Thumbnails
- Viewing history
- Histories containing input filenames or local paths
- Compression or reconstruction results not explicitly saved by the user

The Viewer stores only the selected Japanese / English interface language. This preference contains no image, filename, or path.

When the user chooses Save, Export as PNG, or Copy to Clipboard, the output is written to the selected destination or transferred to the Windows clipboard.

Windows paging and hibernation files, crash dumps, browsers, antivirus software, backup products, clipboard history, and other components outside the Software's control may retain temporary traces. The project does not guarantee that every possible trace can never remain.

When reporting a privacy issue, do not attach private images, filenames, local paths, or sensitive logs to a public issue.
